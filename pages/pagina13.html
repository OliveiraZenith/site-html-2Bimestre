HTML Table

Tabelas em HTML são usadas para exibir dados em formato tabular, com linhas e colunas. Aqui estão dois exemplos de como criar tabelas em HTML:

  1. Tabela simples: Para criar uma tabela básica em HTML, você usa as tags <table> para definir a tabela, <tr> para definir uma linha na tabela, e <td> para definir células de dados dentro das linhas.
  2. Tabela com cabeçalho: Você também pode adicionar um cabeçalho à tabela usando a tag <thead> e células de cabeçalho <th>. Isso ajuda a identificar melhor as colunas.
