HTML SVG

SVG (Scalable Vector Graphics) é uma linguagem baseada em XML para descrever gráficos vetoriais bidimensionais. Ele permite criar gráficos escaláveis e de alta qualidade diretamente no código HTML usando tags SVG. Aqui está uma explicação e um exemplo de uso do SVG em HTML:

  1. Incorporando SVG em HTML: Você pode incorporar gráficos SVG em uma página HTML usando a tag <svg>. Dentro da tag <svg>, você pode usar outras tags SVG para desenhar formas, texto e outros elementos gráficos.
  2. Neste exemplo, estamos desenhando um círculo vermelho, um retângulo azul e um texto "SVG em HTML" dentro de um elemento <svg>.
