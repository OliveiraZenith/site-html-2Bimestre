HTML Quotations

Citações em HTML são usadas para incluir citações ou blocos de texto que são reproduções exatas de outra fonte. Existem duas tags principais para citações em HTML:

  1. <blockquote>: A tag é usada para citar um bloco de texto longo, geralmente de outra fonte. Ela é usada para indicar que o texto é uma citação de outra fonte e geralmente é exibida em uma formatação de bloco separada, como uma indentação.
  2. <q>: A tag é usada para citar um bloco de texto curto, geralmente dentro de um parágrafo. Ela é usada para indicar que o texto é uma citação de outra fonte e geralmente é exibida entre aspas.
