HTML Block & Inline

Em HTML, elementos são categorizados em dois tipos principais: elementos de bloco e elementos inline. Aqui está uma explicação e exemplos de cada tipo:

  1. Elementos de Bloco: Elementos de bloco são aqueles que geralmente começam em uma nova linha e ocupam toda a largura disponível do contêiner pai. Eles são usados para estruturar o layout da página e geralmente contêm outros elementos HTML.Exemplos de elementos de bloco incluem <div>, <p>, <h1> até <h6>, <ul>, <ol>, <table>, entre outros
  2. Elementos Inline: Elementos inline são aqueles que não iniciam em uma nova linha e geralmente ocupam apenas o espaço necessário ao redor de seu conteúdo. Eles são usados para marcar partes do texto ou conteúdo dentro de elementos de bloco.Exemplos de elementos inline incluem <span>, <a>, <strong>, <em>, <img>, <br>, entre outros
