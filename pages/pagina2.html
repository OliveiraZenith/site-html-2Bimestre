HTML Elements

Os elementos HTML são os blocos de construção básicos de uma página web. Cada elemento é representado por uma tag HTML e pode conter texto, outros elementos ou ambos. Aqui estão alguns elementos HTML comuns e exemplos de como eles são usados:

  1. <div>: Define uma divisão ou seção genérica em um documento HTML. É frequentemente usado para agrupar elementos para aplicar estilos CSS ou manipular com JavaScript
  2. <p>: Define um parágrafo de texto
  3. <span>: Define uma pequena seção de texto dentro de um parágrafo ou qualquer outro elemento de bloco
  4. <a>: Define um hyperlink.

  5. <ul>: Define uma lista não ordenada.

  7. <img>: Insere uma imagem na página.
  8. <table>: Define uma tabela
