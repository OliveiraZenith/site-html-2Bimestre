HTML Attributes

Atributos HTML são informações adicionais que podem ser incluídas em tags HTML para fornecer detalhes sobre como um elemento deve ser exibido ou se comportar. Aqui estão alguns exemplos comuns de atributos HTML:

  1. class: Define uma classe CSS para o elemento, permitindo estilização através de CSS.
  2. id: Define um identificador exclusivo para o elemento, que pode ser usado para estilização ou manipulação com JavaScript.
  3. src: Especifica o caminho para o recurso a ser exibido, geralmente usado em elementos como img para especificar o local de uma imagem
  4. href: Especifica o URL de destino para um hyperlink definido pela tag <a>
  5. alt: Fornece um texto alternativo para elementos como , útil para acessibilidade ou quando a imagem não pode ser exibida.
  6. disabled: Desativa um elemento interativo, como um botão ou campo de entrada.
