HTML Colors

As cores em HTML podem ser definidas usando diferentes métodos, como nomes de cores pré-definidos, códigos hexadecimais, códigos RGB ou códigos HSL. Aqui estão dois exemplos de como as cores podem ser definidas em HTML:

  1. Nomes de cores pré-definidos: HTML possui uma lista de nomes de cores pré-definidos que podem ser usados para definir a cor de elementos. Alguns exemplos de nomes de cores incluem "red" (vermelho), "blue" (azul), "green" (verde) e assim por diante.
  2. Códigos hexadecimais: Os códigos hexadecimais são usados para representar cores em HTML usando uma combinação de seis dígitos hexadecimais, precedidos pelo caractere '#'. Cada par de dígitos representa as intensidades de vermelho, verde e azul (RGB) da cor
