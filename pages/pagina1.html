HTML Básico

HTML, ou HyperText Markup Language, é a espinha dorsal da internet. É a linguagem utilizada para criar e estruturar páginas da web. O HTML permite que você organize o conteúdo de uma página, como texto, imagens e vídeos, de forma hierárquica e semântica. Com o HTML básico, você aprenderá os elementos fundamentais para criar uma página web simples, incluindo tags de cabeçalho, parágrafos, listas, links e imagens. Esses conceitos são essenciais para construir uma base sólida antes de avançar para tópicos mais avançados.

1) <!DOCTYPE html>: Define o tipo de documento e a versão do HTML sendo usada. É necessário no início de cada documento HTML.

2) <html>: Define o elemento raiz de um documento HTML.

3) <head>: Contém metadados do documento, como título, links para estilos e scripts, etc.

4) <title>: Define o título da página exibido na barra de título do navegador.

5) <body>: Contém o conteúdo visível da página, como texto, imagens, links, etc.

Exemplo:

        <!DOCTYPE html>
        <html lang="pt-br">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Exemplo de Código HTML</title>
        </head>
        <body>
            <h1>Exemplo de Código HTML</h1>
            <p>Este é um exemplo de como exibir código HTML em uma página web.</p>
        </body>
        </html>
