HTML Comments

Comentários em HTML são usados para fornecer anotações ou informações sobre o código HTML que não devem ser exibidas na página web renderizada pelo navegador. Aqui estão dois exemplos de como os comentários são utilizados em HTML:

  1. Comentários de uma linha: Comentários de uma linha são usados para adicionar anotações breves em uma única linha de código HTML. Eles são delimitados pelos caracteres <!-- e -->.
  2. Comentários de várias linhas: Comentários de várias linhas são usados para adicionar anotações mais longas ou detalhadas que abrangem várias linhas de código HTML. Eles são delimitados por <!-- no início e --> no final, e podem se estender por várias linhas.
